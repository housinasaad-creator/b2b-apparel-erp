# 👨‍💼 دليل ميزات المدير (Admin) - مشغل الأحلام

## ✨ الميزات الحصرية للمدير

المدير فقط يستطيع الوصول إلى ميزتين متقدمتين:

1. **👥 إدارة المستخدمين والمفاتيح**
2. **📊 التقارير والتحليلات الشهرية**

---

## 1️⃣ **إدارة المستخدمين والمفاتيح**

### **الوصول:**
```
من القائمة الجانبية (Sidebar)
↓
👥 إدارة المستخدمين
```

### **الميزات:**

#### **🔑 إنشاء مستخدم جديد**

```
الزر: [+] إضافة مستخدم جديد
     ↓
     نموذج الإنشاء يظهر
```

**البيانات المطلوبة:**

```
1️⃣ الاسم الكامل *
   مثل: أحمد محمد السيد

2️⃣ اسم المستخدم * (Username)
   مثل: ahmed123
   (هذا هو اسم الدخول - بدل البريد)

3️⃣ كلمة المرور *
   مثل: 0000 أو كلمة قوية
   (تنصيح: استخدم أرقام بسيطة للموظفين)

4️⃣ الدور * (3 خيارات)
   ├─ 👷 موظف العمليات (Staff)
   ├─ 📊 مشرف الإنتاج (Manager)
   └─ 👨‍💼 مدير الأحلام (Admin)

5️⃣ إذا اختار "موظف":
   └─ رقم الماكينة * 
      (مثل: 1, 2, 3, 4, 5)
   └─ سيظهر حقل: "رقم الماكينة"
      
6️⃣ إذا اختار "مشرف" أو "مدير":
   └─ القسم (مثل: الإنتاج، الجودة)
```

#### **مثال 1: إضافة موظف جديد**

```
الاسم: محمود على إبراهيم
اسم المستخدم: mahmoud_ali
كلمة المرور: 0000
الدور: 👷 موظف العمليات
رقم الماكينة: 2  ← يُربط الموظف بماكينة رقم 2

تأثير:
├─ الموظف يسجل دخول بـ: mahmoud_ali / 0000
├─ يرى: بيانات الإنتاج فقط
├─ التقارير: تظهر له تقارير ماكينة 2
└─ الأسعار: لا يراها
```

#### **مثال 2: إضافة مشرف جديد**

```
الاسم: فاطمة محمود
اسم المستخدم: fatima_manager
كلمة المرور: 1234
الدور: 📊 مشرف الإنتاج
القسم: إدارة الإنتاج

تأثير:
├─ المشرف يسجل دخول بـ: fatima_manager / 1234
├─ يرى: الأسعار والإحصائيات
├─ يستطيع: تعديل الكميات والأسعار
└─ لا يستطيع: حذف بيانات قديمة
```

#### **مثال 3: إضافة مدير إضافي**

```
الاسم: محمد عبدالعزيز
اسم المستخدم: mohammed_admin
كلمة المرور: admin_pass
الدور: 👨‍💼 مدير الأحلام
القسم: الإدارة العليا

تأثير:
├─ يرى: كل شيء (جميع البيانات)
├─ يستطيع: كل شيء (تعديل، حذف، إضافة)
└─ يمكنه: إضافة مستخدمين آخرين
```

## 2️⃣ **التقارير والتحليلات الشهرية**

### **الوصول:**
```
من القائمة الجانبية (Sidebar)
↓
📊 التقارير الشهرية
```

├─ عدد الآلات بحاجة صيانة
├─ عدد الآلات المعطلة
├─ متوسط OEE لجميع الآلات
├─ إجمالي الإنتاج اليومي
└─ الوقت الذي تم فيه إنشاء التقرير

```

│   └─ يناير، فبراير، ... ديسمبر

├─ إجمالي وقت التوقف

├─ أسوأ ماكينة (الأداء الأقل)
├─ عدد جلسات الصيانة
└─ الوقت

مثال:

```

استخدام التقارير:

## 🎯 **حالات استخدام عملية**

### **حالة 1: إضافة موظف جديد لماكينة معطلة**

```
1. المدير يلاحظ أن ماكينة 3 معطلة
2. يذهب إلى "إدارة المستخدمين"
3. يضغط [+ إضافة مستخدم جديد]
4. يملأ البيانات:
   الاسم: سارة محمود
   اسم المستخدم: sarah_machine3
   كلمة المرور: 0000
   الدور: موظف العمليات
   رقم الماكينة: 3
5. يضغط [✅ إنشاء المستخدم]
6. سارة الآن مسؤولة عن ماكينة 3
```

### **حالة 2: الحصول على تقرير ماكينة قبل لقاء العملاء**

```
1. العميل سيأتي بعد ساعة
2. المدير يريد معلومات عن ماكينة 1
3. يذهب إلى "التقارير الشهرية"
4. يختار: "تقرير الماكينة"
5. يختار الماكينة: ماكينة القص - 1
6. يختار الشهر: يناير 2024
7. يضغط [إنشاء التقرير]
8. يرى التقرير الكامل:
   - 87% OEE ممتاز ✅
   - 2 صيانات فقط
   - ملاحظات إيجابية
9. يضغط [📥 تنزيل PDF]
10. يطبعها ويعطيها للعميل
```

### **حالة 3: تحليل أداء الإنتاج الشهري**

```
1. نهاية الشهر (31 يناير)
2. المدير يريد تحليل الأداء الشامل
3. يذهب إلى "التقارير الشهرية"
4. يختار: "تقرير شهري"
5. يختار: يناير 2024
6. يضغط [إنشاء التقرير الشهري]
7. يرى:
   - إجمالي إنتاج: 1,289,000 قطعة ✅
   - أفضل ماكينة: ماكينة القص (87%)
   - أسوأ ماكينة: ماكينة الخياطة 2 (70%)
8. يتخذ قرارات:
   - تحسين الخياطة 2
   - الحفاظ على أداء القص
9. يصدّر التقرير Excel ويحفظه
```

## 🔐 **أمان المدير**

```
✅ فقط المدير يرى:
  - صفحة إدارة المستخدمين
  - صفحة التقارير الشهرية

❌ المشرف والموظف:
  - لا يرون هذه الصفحات
  - إذا حاولوا الدخول، ينقلون للوحة الرئيسية

💡 التصميم:
  - القائمة الجانبية تظهر فقط الصفحات المسموح بها
  - حماية من الوصول غير المصرح
```

## 🎓 **نصائح للاستخدام الفعال**

```
1️⃣ إضافة المستخدمين:
   ✅ استخدم أسماء وضح (أحمد، محمود، سارة)
   ✅ أسماء مستخدم بسيطة (ahmed, mahmoud, sarah)
   ✅ كلمات مرور سهلة للموظفين (0000)
   ✅ كلمات مرور قوية للمديرين (admin_pass_123)

2️⃣ قراءة التقارير:
   ✅ أنشئ تقرير يومي قبل الظهيرة
   ✅ أنشئ تقرير شهري في نهاية الشهر
   ✅ تابع أسوأ ماكينة بتقرير الماكينة
   ✅ احفظ التقارير كملفات PDF

3️⃣ اتخاذ القرارات:
   ✅ OEE < 75% = تحسين مطلوب
   ✅ OEE 75-85% = جيد
   ✅ OEE >= 85% = ممتاز
```
