## 🔔 نظام التنبيهات والإشعارات

### **أنواع التنبيهات:**

#### **1️⃣ تنبيهات المخزون (Inventory Alerts)**
```
🔴 حرج (Critical):
   - "قماش نايلون انتهى - الكمية: 450 متر (الحد الأدنى: 1000)"
   - إجراء سريع مطلوب

⚠️ منخفض (Low):
   - "خيط أسود بحاجة إعادة طلب - الكمية: 30 بكرة"
   - إجراء تحضير طلب
```

#### **2️⃣ تنبيهات الآلات (Machine Alerts)**
```
🔧 صيانة (Maintenance):
   - "ماكينة الخياطة - 2 بحاجة صيانة دورية"
   - جدولة الصيانة

📊 كفاءة (Efficiency):
   - "ماكينة الخياطة - 1 كفاءتها انخفضت إلى 60%"
   - فحص سريع
```

#### **3️⃣ تنبيهات الطلبيات (Order Alerts)**
```
✅ مكتملة (Completed):
   - "الطلبية PO-2024-001 اكتملت بنجاح"
   - جاهزة للشحن

⏰ متأخرة (Delayed):
   - "الطلبية PO-2024-002 متأخرة عن الموعد المحدد"
   - تحتاج متابعة
```

### **كيفية استخدام التنبيهات:**

```
1. اضغط على 🔔 في أعلى اليمين
2. شاهد جميع التنبيهات والإشعارات
3. أرقام حمراء = تنبيهات غير مقروءة
4. اضغط "قراءة" لتحديث الإشعار
5. اضغط "إغلاق" لحذف الإشعار
6. اضغط "عرض" للذهاب إلى الصفحة المتعلقة
```

## 🎯 الميزات الرئيسية

### **1️⃣ المصادقة الآمنة**
- تسجيل دخول آمن
- تخزين بيانات المستخدم محلياً
- تسجيل خروج فوري
- حماية الصفحات من الوصول غير المصرح

### **2️⃣ الأدوار والصلاحيات**
- كل دور له صلاحيات مختلفة
- عرض/إخفاء الميزات حسب الدور
- قيود على العمليات الحساسة
- تتبع من فعل ماذا

### **3️⃣ التنبيهات والإشعارات**
- تنبيهات فورية
- أنواع مختلفة (خطير، تحذير، نجاح، معلومة)
- عداد للتنبيهات غير المقروءة
- ربط مباشر إلى الصفحة المتعلقة

### **4️⃣ تجربة المستخدم**
- واجهة بسيطة وواضحة
- تصميم احترافي
- رسائل خطأ مفيدة
- تعليمات سهلة الفهم
