# 🏭 النظام المتكامل - شرح كامل

## 📋 السيناريو العملي الكامل

### المثال: مدير مصنع يريد إضافة طلبية قمصان

```
المدير يقول: "بدي أضيف طلبية 1500 قميص أحمر"
```

---

## 🎯 الخطوات الكاملة (من البداية للنهاية)

### **الخطوة 1️⃣: اضغط على "إضافة طلبية جديدة"**

```
الصفحة: خط الإنتاج والتصنيع
→ الزر الأخضر: "إضافة طلبية جديدة"
```

يفتح لك نموذج متقدم بـ 6 أقسام:

---

### **الخطوة 2️⃣: أدخل معلومات الطلبية الأساسية**

```
📝 القسم الأول: معلومات المنتج
┌─────────────────────────────────────────┐
│ اسم المنتج: قمصان رجالية أحمر           │
│ الكمية المطلوبة: 1500 قطعة              │
│ الوحدة: قطعة                            │
└─────────────────────────────────────────┘
```

---

### **الخطوة 3️⃣: أدخل السعر والحسابات التلقائية**

```
💵 القسم الثاني: التسعير والإيرادات
┌──────────────────────────────────────────┐
│ سعر البيع للقطعة: $25                    │
├──────────────────────────────────────────┤
│ الإيراد الإجمالي: $37,500  ✅ محسوب     │
│ (تلقائياً: 25 × 1500)                   │
├──────────────────────────────────────────┤
│ الربح المتوقع: يحدّث مع إضافة المواد    │
└──────────────────────────────────────────┘
```

---

### **الخطوة 4️⃣: أدخل التواريخ والمرحلة**

```
📅 القسم الثالث: الجدول الزمني
┌──────────────────────────────────────────┐
│ تاريخ البدء: 2024-07-05                  │
│ التاريخ المتوقع للانتهاء: 2024-07-12    │
│ المرحلة الحالية: قيد الانتظار ▼          │
│ نسبة التقدم: 0% [━━━━━━━━━━]             │
└──────────────────────────────────────────┘
```

---

### **الخطوة 5️⃣: أضف المواد المستخدمة** ⭐ الأهم!

```
📦 القسم الرابع: المواد المستخدمة
┌──────────────────────────────────────────┐
│ اسم المادة: قماش قطني                    │
│ الكمية: 2250 متر                        │
│ الوحدة: متر ▼                           │
│ [إضافة] ←── زر الإضافة                  │
└──────────────────────────────────────────┘

👇 بعد الضغط على إضافة:

المواد المضافة:
┌──────────────────────────────────────────┐
│ • قماش قطني: 2250 متر              [حذف] │
└──────────────────────────────────────────┘

✏️ أضف مادة ثانية:
┌──────────────────────────────────────────┐
│ اسم المادة: خيط أحمر                    │
│ الكمية: 9000 بكرة                       │
│ الوحدة: بكرة ▼                          │
│ [إضافة] ←── زر الإضافة                  │
└──────────────────────────────────────────┘

👇 بعد الضغط على إضافة:

المواد المضافة:
┌──────────────────────────────────────────┐
│ • قماش قطني: 2250 متر              [حذف] │
│ • خيط أحمر: 9000 بكرة               [حذف] │
└──────────────────────────────────────────┘

✏️ أضف مادة ثالثة:
┌──────────────────────────────────────────┐
│ اسم المادة: أزرار بلاستيكية              │
│ الكمية: 6000 قطعة                       │
│ الوحدة: قطعة ▼                          │
│ [إضافة] ←── زر الإضافة                  │
└──────────────────────────────────────────┘

✅ النتيجة النهائية:
┌──────────────────────────────────────────┐
│ المواد المضافة:                          │
│ ✓ قماش قطني: 2250 متر                   │
│ ✓ خيط أحمر: 9000 بكرة                    │
│ ✓ أزرار بلاستيكية: 6000 قطعة             │
└──────────────────────────────────────────┘
```

---

### **الخطوة 6️⃣: اضغط "إضافة الطلبية"**

```
عند الضغط على الزر، النظام يفعل:

1️⃣ التحقق من المخزون:
   ✓ قماش قطني: متوفر 8500 متر ← نحتاج 2250 ✅
   ✓ خيط أحمر: متوفر 125 بكرة ← نحتاج 9000 ❌ تحذير!
   ✓ أزرار: متوفر 15000 قطعة ← نحتاج 6000 ✅

2️⃣ إذا كان كل شيء تمام:
   ✓ إضافة الطلبية
   ✓ طرح المواد من المخزون
   ✓ حساب الأرباح

3️⃣ رسالة نجاح:
   ✅ "تمت إضافة الطلبية بنجاح!"
   ✅ "تم طرح المواد من المخزون"
```

## 🎮 الميزات الديناميكية

### **1️⃣ تعديل الكمية الجاهزة مباشرة في الجدول**

```
اضغط على الكمية في الجدول:
┌────────────┐
│ 1500 قطعة  │ ← يصير حقل قابل للتعديل
└────────────┘

غيّر الرقم إلى 2000:
┌────────────┐
│ 2000 قطعة  │
└────────────┘

✅ يتحدث الإجمالي تلقائياً إلى $50,000
```

### **2️⃣ الحسابات تتحدث فوراً**

```
عند إضافة المواد → الإيراد يتحدث
عند تغيير السعر → الربح يتحدث
عند تغيير الكمية → الإجمالي يتحدث
```

## 📈 تأثير على المخزون والإحصائيات

### **عند إضافة الطلبية:**

```
المخزون قبل الطلبية:
┌──────────────────────┐
│ قماش قطني: 8500 متر │
│ خيط أحمر: 125 بكرة  │
│ أزرار: 15000 قطعة  │
└──────────────────────┘

         ↓ بعد إضافة الطلبية ↓

المخزون بعد الطلبية:
┌──────────────────────┐
│ قماش قطني: 6250 متر │ ✓ -2250
│ خيط أحمر: ??? بكرة  │ ❌ لا توجد كمية
│ أزرار: 9000 قطعة   │ ✓ -6000
└──────────────────────┘

⚠️ تنبيه في صفحة المخازن:
الخيط الأحمر بحالة حرجة!
```

### **الإحصائيات:**

```
لوحة التحكم تحدّث:
┌─────────────────────────────────┐
│ إجمالي الطلبيات: 6 طلبيات      │ ✓ +1
│ القيمة الإجمالية: $xxxxx       │ ✓ +37,500
│ قيد الإنتاج: x طلبيات          │ ✓ قد يزيد
└─────────────────────────────────┘
```

## 🔄 تعديل الطلبية الموجودة

### **الخطوات:**

```
1️⃣ ابحث عن الطلبية في الجدول
2️⃣ اضغط على أيقونة التعديل (قلم أزرق)
3️⃣ النموذج يفتح مع البيانات الحالية
4️⃣ عدّل ما تريد (السعر، المواد، المراحل)
5️⃣ اضغط "تحديث الطلبية"
```

---

## 🗑️ حذف الطلبية

### **ما يحدث:**

```
عند حذف الطلبية:

1️⃣ يتم حذف الطلبية من الجدول
2️⃣ يتم إرجاع المواد للمخزون تلقائياً:
   • قماش قطني: +2250 متر
   • خيط أحمر: +9000 بكرة
   • أزرار: +6000 قطعة

3️⃣ رسالة تأكيد:
   ✅ "تم حذف الطلبية وإرجاع المواد للمخزون"
```

## 🎯 مثال عملي شامل (من البداية للنهاية)

### **الحالة الأولية:**
```
المخزون:
- قماش قطني: 8500 متر
- خيط: 125 بكرة
- أزرار: 15000 قطعة

الطلبيات: 5 طلبيات فقط
```

### **الإجراء:**
```
1. المدير يفتح الموقع
2. يذهب إلى "خط الإنتاج والتصنيع"
3. يضغط "إضافة طلبية جديدة"
4. يملأ البيانات:
   - المنتج: قمصان أحمر
   - الكمية: 1500 قطعة
   - السعر: $25
   - المواد: قماش 2250م، خيط 9000، أزرار 6000
5. يضغط "إضافة الطلبية"
```

### **النتيجة:**
```
✅ النظام يفعل كل هذا تلقائياً:

1. ✓ يتحقق من المخزون
2. ✓ يطرح المواد من المخزون
3. ✓ يضيف الطلبية للجدول
4. ✓ يحسب الإيراد: $37,500
5. ✓ يحسب الربح المتوقع
6. ✓ يحدّث الإحصائيات
7. ✓ يعطي تنبيهات إذا كانت هناك مشاكل
8. ✓ يعرض رسالة نجاح

الآن الحالة:
المخزون:
- قماش قطني: 6250 متر ✓ -2250
- خيط: ??? بكرة ❌ مشكلة!
- أزرار: 9000 قطعة ✓ -6000

الطلبيات: 6 طلبيات
القيمة الإجمالية: $xxxxx + $37,500
```

✅ إضافة طلبيات بكل تفاصيلها
✅ إدارة المواد المستخدمة
✅ التحقق من المخزون تلقائياً
✅ تحديث المخزون فوراً
✅ حساب الأرباح والإيرادات
✅ إصدار تنبيهات ذكية
✅ عرض كل التفاصيل في الجدول
✅ تعديل الأرقام بسهولة
✅ حذف آمن مع إرجاع المواد

**من الإضافة إلى النهاية - كل شيء مرتبط ومتكامل! 🎉**
