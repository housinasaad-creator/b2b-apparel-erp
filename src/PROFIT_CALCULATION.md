# 💰 شرح حساب الأرباح والإيرادات في نظام ERP

## 🎯 المعادلة الأساسية

```
الربح اليومي = الإيرادات الصافية - التكاليف الإجمالية
Daily Profit = Net Revenue - Total Costs
```

---

## 📊 تفصيل المعادلة خطوة بخطوة

### ✅ الخطوة 1: حساب الإيرادات الإجمالية

**الصيغة:**
```
الإيرادات الإجمالية = سعر البيع × عدد القطع المنتجة
Total Revenue = Price Per Unit × Daily Production
```

**مثال عملي:**
```
سعر القطعة الواحدة: $25
الإنتاج اليومي: 45,320 قطعة

الإيرادات = 25 × 45,320 = $1,133,000
```

---

### ✅ الخطوة 2: حساب الخصم (إن وجد)

**الصيغة:**
```
الخصم = الإيرادات الإجمالية × نسبة الخصم
Discount = Total Revenue × Discount %
```

**مثال عملي:**
```
الإيرادات الإجمالية: $1,133,000
نسبة الخصم: 5%

الخصم = 1,133,000 × 5% = $56,650
```

---

### ✅ الخطوة 3: حساب الإيرادات الصافية

**الصيغة:**
```
الإيرادات الصافية = الإيرادات الإجمالية - الخصم
Net Revenue = Total Revenue - Discount
```

**مثال عملي:**
```
الإيرادات الإجمالية: $1,133,000
الخصم: $56,650

الإيرادات الصافية = 1,133,000 - 56,650 = $1,076,350
```

---

### ✅ الخطوة 4: حساب التكاليف الإجمالية

التكاليف تتضمن **ثلاثة أنواع:**

#### أ) تكلفة المواد الخام (Raw Materials)
```
المواد الخام = قماش + خيوط + أزرار + مواد أخرى
Raw Materials = Fabric + Threads + Buttons + Other Materials
```
**مثال:** $5,000 يومياً

#### ب) تكلفة العمالة (Labor Cost)
```
العمالة = رواتب العمال + مكافآت + تأمين صحي
Labor = Wages + Bonuses + Benefits
```
**مثال:** $2,000 يومياً

#### ج) النفقات العامة والتشغيلية (Operational Overhead)
```
النفقات = الكهرباء + الصيانة + الإيجار + الماء + الغاز + التأمين
Overhead = Electricity + Maintenance + Rent + Water + Gas + Insurance
```
**مثال:** $800 يومياً

**التكاليف الإجمالية:**
```
التكاليف الكلية = المواد + العمالة + النفقات
Total Costs = 5,000 + 2,000 + 800 = $7,800
```

---

### ✅ الخطوة 5: حساب الربح النهائي

**الصيغة:**
```
الربح = الإيرادات الصافية - التكاليف الإجمالية
Profit = Net Revenue - Total Costs
```

**مثال عملي:**
```
الإيرادات الصافية: $1,076,350
التكاليف الإجمالية: $7,800

الربح اليومي = 1,076,350 - 7,800 = $1,068,550 ✅
```

---

## 📈 حساب نسب مهمة

### 1) نسبة الربح الصافي (Profit Margin)

**الصيغة:**
```
نسبة الربح = (الربح / الإيرادات الصافية) × 100
Profit Margin = (Profit / Net Revenue) × 100
```

**مثال:**
```
الربح: $1,068,550
الإيرادات الصافية: $1,076,350

نسبة الربح = (1,068,550 / 1,076,350) × 100 = 99.27% 📈
```

---

### 2) تكلفة القطعة الواحدة

**الصيغة:**
```
تكلفة القطعة = التكاليف الإجمالية ÷ عدد القطع
Cost Per Unit = Total Costs ÷ Daily Production
```

**مثال:**
```
التكاليف: $7,800
الإنتاج: 45,320 قطعة

تكلفة القطعة = 7,800 ÷ 45,320 = $0.17 لكل قطعة
```

---

### 3) ربح القطعة الواحدة

**الصيغة:**
```
ربح القطعة = (الإيراد - التكلفة) للقطعة الواحدة
Profit Per Unit = Revenue Per Unit - Cost Per Unit
```

**مثال:**
```
إيراد القطعة: $25
تكلفة القطعة: $0.17

ربح القطعة = 25 - 0.17 = $24.83 لكل قطعة
```

---

## 🔄 مثال شامل متكامل

### البيانات المدخلة:
| البند | القيمة |
|------|--------|
| سعر البيع (القطعة) | $25 |
| الإنتاج اليومي | 45,320 قطعة |
| تكلفة المواد الخام | $5,000 |
| تكلفة العمالة | $2,000 |
| النفقات التشغيلية | $800 |
| نسبة الخصم | 5% |

### الحسابات:
```
1️⃣ الإيرادات الإجمالية
   = $25 × 45,320 = $1,133,000

2️⃣ الخصم المطبق
   = $1,133,000 × 5% = $56,650

3️⃣ الإيرادات الصافية
   = $1,133,000 - $56,650 = $1,076,350

4️⃣ التكاليف الإجمالية
   = $5,000 + $2,000 + $800 = $7,800

5️⃣ الربح اليومي
   = $1,076,350 - $7,800 = $1,068,550 ✅

6️⃣ نسبة الربح
   = ($1,068,550 ÷ $1,076,350) × 100 = 99.27% 📈

7️⃣ تكلفة القطعة
   = $7,800 ÷ 45,320 = $0.17

8️⃣ ربح القطعة
   = $25 - $0.17 = $24.83
```

## 🎚️ كيفية تعديل البيانات

1. **اذهب إلى الإعدادات** من القائمة الجانبية
2. **عدّل القيم التالية:**
   - سعر البيع للقطعة
   - الإنتاج اليومي
   - تكلفة المواد الخام
   - تكلفة العمالة
   - النفقات التشغيلية
   - نسبة الخصم

3. **شاهد الحسابات تتحدث فوراً!**

## 💡 نصائح مهمة

### لزيادة الربح:
1. **↑ زيادة سعر البيع** (إذا كان السوق يقبله)
2. **↑ زيادة الإنتاج** (الاستفادة من الطاقة الإنتاجية)
3. **↓ تقليل التكاليف** (تحسين كفاءة الآلات)
4. **↓ تقليل الخصم** (تقليل العروض والخصومات)

### مثال التحسن:
```
الوضع الحالي:
- الإنتاج: 45,320 قطعة
- الربح: $1,068,550

لو زادت الإنتاج بـ 10%:
- الإنتاج: 49,852 قطعة
- الربح الجديد: ~$1,177,435
- الزيادة: $108,885 إضافي ✅
```
